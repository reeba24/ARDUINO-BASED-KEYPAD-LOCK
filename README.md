# ARDUINO-BASED-KEYPAD-LOCK
A project on a password-based door lock security system using Arduino and a keypad.

SOFTWARE REQUIREMENTS:
1. Arduino 1.8.19

HARDWARE REQUIREMENTS:
1. 220 Ohms Resistor 
2. Jumper Wires 
3. LEDs 
4. Servo Motor 
5. Arduino UNO 
6. Keypad 
7. Breadboard
